# Exercise 01: Connect declarative agent to MCP server

## Introduction
In this exercise, you'll run a complete Model Context Protocol (MCP) server for Zava Insurance's claims system and integrate it with declarative agent in Microsoft 365 Copilot that you'll create, enabling natural language interactions with real claims data through secure, standardized AI agent communication.
