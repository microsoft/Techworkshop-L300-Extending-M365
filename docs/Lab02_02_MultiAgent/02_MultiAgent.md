# Exercise 02: Multi-agent claims orchestration

## Introduction
In this exercise, you'll create a declarative agent in Microsoft 365 Copilot that leverages embedded knowledge-local files containing Zava Insurance's contractor pricing data and claims inspection guidelines. 

You'll then build Zava Care, a Connected Agent that orchestrates your specialized agents (Zava Claims and Zava Procurement) into a unified claims processing experience, enabling insurance adjusters to seamlessly access claims data, procurement information, and cost estimates through natural language interactions.
